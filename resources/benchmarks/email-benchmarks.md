# Cold Email Benchmarks

Performance data for cold email campaigns. Use these to set expectations,
diagnose issues, and validate results.

## Reply Rates by Approach

| Approach | Reply Rate | Context |
|----------|-----------|---------|
| Cold, no signal, no personalization | 1-3% | Generic outbound |
| ICP-segmented, no signal | 4-6% | Segmented but no timing |
| Single signal, personalized | 18-22% | One trigger, relevant message |
| Multi-signal, 3+ stacked | 35-40% | Compound signals, full personalization |

## Reply Rates by Sequence Position

| Position | Reply Rate % of Total | Purpose |
|----------|----------------------|---------|
| Email 1 | 55-65% | Introduce, establish relevance |
| Email 2 | 20-25% | New angle, add context |
| Email 3 | 10-20% | Lower friction, accept the no |

## Subject Line Performance

| Type | Open Rate Lift | Risk |
|------|---------------|------|
| Name + company | Baseline | Low |
| Question format | +5-10% | Can feel clickbait |
| Signal reference | +10-15% | Requires accurate data |
| Mutual connection | +15-20% | Requires real connection |
| One word | +5-8% | Curiosity play, can miss |

## Word Count Impact

| Word Count | Reply Rate Trend |
|------------|-----------------|
| 25-50 words | Highest reply rate for ATL |
| 50-90 words | Optimal range for most segments |
| 90-120 words | Acceptable for BTL with detail |
| 120+ words | Sharp decline in reply rate |

## Deliverability Thresholds

| Metric | Healthy | Warning | Critical |
|--------|---------|---------|----------|
| Bounce rate | < 1% | 1-2% | > 2% |
| Complaint rate | < 0.05% | 0.05-0.1% | > 0.1% |
| Open rate | > 60% | 40-60% | < 40% |
| Spam folder rate | < 5% | 5-15% | > 15% |

## Timing Data

| Factor | Optimal |
|--------|---------|
| Days between emails | 3-5 days |
| Best send days | Tuesday, Wednesday, Thursday |
| Best send time | 8-10 AM recipient timezone |
| Signal freshness (job change) | Days 14-45 |
| Signal freshness (funding) | Weeks 1-4 |
| Signal freshness (website visit) | Within 24 hours |
| Re-verification cadence | Every 30 days |

|--------|-----------|

| Warmup period | 4-8 weeks |
| Email format | Plain text only |
